Returns the list of groups to which the launching of a tool is restricted

getToolGroupsRestriction(string $toolid, string $instance) : array
static
FIXME

: This should really be a part of the above function, getToolGroups. The problem seems to be related to the #tool_groups table. If there is a tool that was at one point restricted to a group, but it isn't anymore, there remains a #__tool_groups entry. The above function will grab that restriction and undoly say that the tool is restricted to a group. The solution seems to be removing the #tool_groups entry if the tool is changed to no longer be restricted to a group. At the moment, this function also checks to ensure that toolaccess='@GROUP' from the #__tool_version table

Arguments

$toolid

string

is the tool id

$instance

string

is the tool version instance

Response

array

Return list of groups
